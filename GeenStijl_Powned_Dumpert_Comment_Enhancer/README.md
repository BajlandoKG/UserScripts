## Description

Add features to enhance comments on GeenStijl & Powned & Dumpert & more.

Currently supported sites:

* http://geenstijl.nl
* http://geenstijl.tv
* http://powned.tv
* http://dumpert.nl
* http://daskapital.nl
* http://glamora.ma
